## YouTube Video RAG System

An AI-powered Retrieval-Augmented Generation (RAG) system that extracts YouTube video transcripts, converts them into embeddings using LangChain, stores them in a FAISS vector database, and enables context-aware question answering from video content.

### Tech Stack
- Python
- LangChain
- FAISS
- YouTube Transcript API
- LLMs
- Vector Embeddings
